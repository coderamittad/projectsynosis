Who is team lead?

Prashant Shete is the team lead for the Gaming Website project, overseeing its development, managing the team, and ensuring successful execution. He is responsible for task delegation, coordination, and maintaining smooth communication throughout the project.

project intro?

The project is a Gaming Website aimed at providing gaming content, community interaction, and a dynamic user experience. It features seamless navigation, robust backend services, and an engaging platform for gamers to connect, explore, and enjoy content.

Ajay Shinde

Ajay Shinde, as a Full Stack Developer, handles both front-end and back-end development. He works on user interfaces, server-side logic, and database management, ensuring seamless functionality and high performance of the website.


Amit Tad

Amit Tad, specializing in Spring Boot and Thymeleaf, focuses on backend services for the project. He develops and maintains server-side logic using Spring Boot, while leveraging Thymeleaf for rendering dynamic web pages. His work ensures efficient backend performance, secure data handling, and smooth integration with the front-end, contributing to a seamless user experience.
